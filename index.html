Apa itu Fungsi Kuadrat?

Fungsi kuadrat adalah fungsi polinomial berderajat dua. Ada tiga bentuk representasinya:

Bentuk Umum (Standard Form)

f(x) = ax² + bx + c

Bentuk Vertex (Vertex Form)

f(x) = a(x − h)² + k

h = absis vertex, k = ordinat vertex

Bentuk Faktor (Factored Form)

f(x) = a(x − p)(x − q)

p, q = akar-akar persamaan

Karakteristik Grafik

1

Arah Bukaan

a > 0 → terbuka ke atas (nilai minimum). a < 0 → terbuka ke bawah (nilai maksimum).

2

Titik Puncak & Sumbu Simetri

Vertex: (-b/2a, f(-b/2a))  |  Sumbu simetri: x = -b/2a

3

Diskriminan & Akar

D > 0: dua akar berbeda  |  D = 0: satu akar kembar  |  D < 0: tidak ada akar real

4

Konversi Bentuk

Lengkapi kuadrat untuk mendapat bentuk vertex. Faktorkan untuk bentuk faktor.

Rumus-Rumus Penting

Titik Puncak

x = -b/2a

Diskriminan

D = b² − 4ac

Rumus ABC

x = (−b ± √D)/2a

Nilai Optimum

y = (4ac − b²)/4a

Aplikasi di Kehidupan Nyata

⚽

Lintasan Bola

Ketika bola ditendang, lintasannya membentuk parabola: h(t) = −5t² + 20t + 1

h = tinggi, t = waktu (detik)

💰

Keuntungan Bisnis

Profit toko bergantung harga jual: P(x) = −2x² + 80x − 300

Cari x agar profit maksimum!

🏗️

Jembatan Lengkung

Desain busur jembatan menggunakan fungsi kuadrat: y = −0.1x² + 4x

Tentukan titik tertinggi busur!

📡

Antena Parabola

Permukaan antena berbentuk parabola untuk memfokuskan sinyal ke satu titik (fokus).

Fisika & telekomunikasi

🌾

Optimasi Lahan

Luas pagar dengan keliling tetap: L(x) = x(100 − x) = −x² + 100x

Maksimalkan luas dengan x = 50!

🔭

Orbit Benda Langit

Lintasan komet dan benda langit tertentu mendekati bentuk parabola saat melewati matahari.
